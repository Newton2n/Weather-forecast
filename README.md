# Weather Forecast App

A simple weather forecasting web application that displays the current weather, hourly forecasts, and a 7-day outlook for any city.  
Built using the **Open-Meteo API** for both geocoding and weather data, with a responsive UI and light/dark theme support.

## 🌐 Live Demo
[**View Live App**](https://newton2n.github.io/Weather-forecast/)

---

## ✨ Features

- **Search by City** – Get weather details for any city using Open-Meteo geocoding.
- **Current Weather** – Displays temperature, "feels like" temperature, wind speed, UV index, and chance of rain.
- **Hourly Forecast** – Shows the next 6 hours with temperature, time, and weather icons.
- **7-Day Forecast** – Displays max/min temperature, weather description, and icons for the week.
- **Dynamic Icons** – Icons change automatically based on weather codes.
- **Light/Dark Theme** – Toggle between modes, with your choice saved in `localStorage`.
